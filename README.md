## 使用

- 下载 [node](http://nodejs.cn/download/)

- 安装 [cnpm](http://npm.taobao.org/)

```
$ npm install -g cnpm --registry=https://registry.npm.taobao.org
```
- 下载项目
```
git clone https://github.com/yunyi1895/docTemplate.git
```
- 安装依赖
```
cnpm run i
```
- 本地启动项目进行预览
```
npm run dev
```

- 打开 http://localhost:5566

## 主要事项

- 发布之前一定要重命名文件夹

doc改成项目名称

- 发布到服务器

```
npm run push
```
